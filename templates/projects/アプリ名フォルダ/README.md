# アプリ名フォルダ

このフォルダは、新しく作成するアプリ用の雛形です。

実際に使うときは、`アプリ名フォルダ` を作成したいアプリ名に置き換えてください。

例:

```text
projects/my-app/README.md
projects/receipt-fridge-ai/README.md
projects/gameapp/README.md
```

## フォルダ作成ルール

新しいアプリを作る場合は、最初に以下を作成する。

```text
projects/<アプリ名>/README.md
```

README.mdを作成してから、必要に応じてフロントエンド、バックエンド、ドキュメントなどのフォルダを追加する。
